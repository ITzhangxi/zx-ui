# Toast
# 属性
| 名称 | 类型 | 默认值 | 说明 | 是否必传 | 
| ---- | ---- | ----- | ----- | ----- |
| time | Number | 2000ms | 显示时间| 否 | 
| icon | String | —— | 字体图标| 否 | 
| iconColor | String | #ffffff | 字体图标颜色| 否 | 
| text | String | —— | 展示的内容| 否 | 
| textColor | String | #ffffff | 提示内容的样色| 否 | 
| contentColor | String | #ffffff | 内容的颜色| 否 | 
| isSHowMask | Boolean | #ffffff | 是否显示遮罩| 否 | 
| type | String | —— | 显示类型 | 否 | 

### type 属性
| 名称 | 说明 |
| ---- | ---- |
| success | 成功 |
| fail | 失败 |
| cancel | 取消 |
| warning | 警告 |
